# IPL 2022 Data Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the IPL 2022 dataset to uncover insights about team performance, match outcomes, player achievements, and match strategies.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset Features
- Match date
- Teams playing
- Toss winner and decision
- Match winner
- Top scorer and highest score
- Best bowling figures

## Key Insights
- Teams with the highest number of wins
- Players with the highest individual scores
- Best bowling performances in matches
- Impact of toss decision on match outcomes
- Match margin analysis

## Conclusion
The analysis highlights key patterns in IPL 2022 matches, including player performance trends, team strategies, and match-winning factors.
